# 🎮 Meus Jogos em Java 

Bem-vindo(a) ao meu repositório de jogos! Aqui está uma coleção de clássicos e desafios de tabuleiro que desenvolvi **do zero e finalizei por completo** utilizando a linguagem Java. 


---

## 🕹️ Jogos Desenvolvidos

### ♟️ Projeto Xadrez
*Um sistema completo de xadrez jogável pelo terminal/console (ou interface gráfica, caso tenha feito), aplicando regras complexas como xeque, xeque-mate e jogadas especiais.*
* **Destaques:** Alta aplicação de Programação Orientada a Objetos (POO), matrizes e lógica de estados.
* **Código-fonte:** [Acessar pasta do projeto](https://github.com/Boueri/Jogos/tree/master/ProjetoXadrez)

### 🏓 Pong
*O clássico jogo de arcade que iniciou a indústria dos games, com movimentação de paletas e rebote de bola com aceleração.*
* **Destaques:** Lógica de colisão física simples e inteligência artificial básica para o oponente.
* **Código-fonte:** [Acessar pasta do projeto](https://github.com/Boueri/Jogos/tree/master/pong)

### 🐍 Snake Xenzia & Snake Clássico
*Duas versões do famoso jogo da cobrinha. O **Snake Clássico** e o **Snake Xenzia**, trazendo a nostalgia dos celulares antigos com gerenciamento de pontuação e colisão com o próprio corpo.*
* **Destaques:** Controle de Game Loop, crescimento dinâmico da estrutura (Listas/Filas) e geração aleatória de comida.
* **Código-fonte:** [Acessar pasta do projeto 1](https://github.com/Boueri/Jogos/tree/master/snake%20xenzia)[/Acessar pasta do projeto 2](https://github.com/Boueri/Jogos/tree/master/snake)

### 👾 Space Invaders
*Um jogo de tiro espacial (Shoot 'em up) onde o jogador deve defender a terra de hordas de alienígenas que descem a tela.*
* **Destaques:** Renderização de múltiplos objetos na tela, sistemas de tiro/projéteis e gerenciamento de vidas.
* **Código-fonte:** [Acessar pasta do projeto](https://github.com/Boueri/Jogos/tree/master/space%20invaders)

### 🧱 Tetris
*O quebra-cabeça mais famoso do mundo, onde blocos geométricos caem e o jogador precisa completar linhas horizontais para eliminá-los.*
* **Destaques:** Lógica avançada de rotação de matrizes (tetraminós) e verificação de linhas completas.
* **Código-fonte:** [Acessar pasta do projeto](https://github.com/Boueri/Jogos/tree/master/tetris)

# 📖 O que aprendi

Durante o desenvolvimento deste projeto, consolidei conhecimentos importantes em desenvolvimento de software, entre eles:

- ☕ Programação Orientada a Objetos (POO)
- 🏗️ Organização e estruturação de projetos Java
- 🧩 Modelagem de classes e objetos
- 🔄 Encapsulamento, Herança e Polimorfismo
- 🎨 Desenvolvimento de interfaces gráficas com Swing
- 🖱️ Tratamento de eventos do teclado e do mouse
- 📐 Implementação de algoritmos e lógica de programação
- 📊 Manipulação de estruturas de dados e coleções
- 🐞 Depuração (debug) e resolução de problemas
- ⚡ Otimização e refatoração de código
- 📂 Organização de recursos (imagens, sons e arquivos)
- 🌿 Versionamento de código com Git e GitHub
- 📖 Documentação técnica utilizando Markdown
- 💡 Planejamento e desenvolvimento de um projeto do início ao fim
## 🛠️ Conceitos de Programação Aplicados

Desenvolver esses jogos me permitiu dominar conceitos fundamentais que vão muito além do desenvolvimento de games:

* **Estrutura de Dados & Matrizes:** Essenciais para o tabuleiro do Xadrez e o grid do Tetris.
* **Programação Orientada a Objetos (POO):** Encapsulamento, Herança e Polimorfismo para gerenciar entidades (jogadores, inimigos, peças).
* **Game Loop & Renderização:** Controle de frames por segundo (FPS) e atualização de lógica em tempo real (Pong, Snake, Space Invaders).
